# Estructura del Repositorio

Este repositorio contiene una organización básica de datos para prácticas de GitHub.  
Aquí se almacenan archivos CSV relacionados con ventas y clientes, organizados en carpetas para mantener un orden claro.

## Contenido principal

- **datos_ventas/**  
  Carpeta donde se almacena información relacionada a las ventas del negocio.

  - **ventas.csv**  
    Contiene datos simples como número de venta, producto y monto.

  - **datos_clientes/**  
    Subcarpeta con información adicional del cliente.
    
    - **clientes.csv**  
      Registra información básica de los clientes.

## Conclusión
Este repositorio demuestra una estructura clara y ordenada, organizada por carpetas y archivos en formato CSV. Además, se utiliza Markdown para documentar la información de manera sencilla y legible, como se aprende en primer semestre de Ingeniería en Software.
